About TaraLabs.3D

TaraLabs.3D is a small Indonesian store specializing in 3D-printed products — articulated figures, anime collectibles, home decor, custom keychains, and made-to-order prints. As our catalog grows, we built TaraLabs, an internal tool that talks to the Shopee Open Platform API so we can manage listings, stock, pricing, and orders more efficiently. This application is used solely by our own shop and is not distributed to or used by any other merchants.

How it works

A lightweight, secure integration with the Shopee Open Platform.

Authorize

The shop owner authorizes the app through Shopee’s official OAuth flow.

Sync

The app reads products and orders from Shopee using the official API v2 endpoints.

Manage

Edits made in our spreadsheet dashboard are pushed back to Shopee — stock, price, and listings.

Monitor

Orders and inventory stay in sync so nothing is oversold or missed.
